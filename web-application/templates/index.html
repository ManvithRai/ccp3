  • File Directory

Instructions

  • User can upload the file to the cloud in a secured manner.

  • User can choose from the list of all the different files that are present on the cloud provided he has the required credentials for decrypting the file.

  • User can download public key associated with the user with whom the file transfer is to take place.

  • To upload the file to cloud storage the user needs to be registered so that his file can be shared with the other desired person in an instant.
